A comprehensive fitness tracking and analytics plugin with achievement system and aboriginal language support.

== Description ==

MKWA Fitness is a WordPress plugin designed to help track fitness activities, manage achievements, and provide detailed analytics. It includes features for point tracking, achievements, and community engagement with aboriginal language support.

Key Features:

* Activity tracking and points system
* Achievement badges and levels
* Analytics dashboard
* Community goals
* Aboriginal language support
* Peak/Off-peak time management
* Leaderboard system

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/mkwa-fitness`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings via the MKWA Fitness menu in the admin dashboard

== Frequently Asked Questions ==

= How does the points system work? =

Points are awarded for various activities:
* Gym visits: 10 points
* Class attendance: 20 points
* Referrals: 50 points
* Streak bonuses apply after 3 consecutive days

= Is the plugin GDPR compliant? =

Yes, the plugin only stores necessary fitness and achievement data with user consent.
